# neetcode-viz

A visualization tool for NeetCode problems and solutions.  
This project aims to help users better understand algorithms and data structures through interactive visualizations.

## Features

- Visualize data structures and algorithms.
- Explore various NeetCode problems interactively.
- Intuitive and user-friendly interface.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/phamthanhhang208/neetcode-viz.git
   ```
2. Install dependencies (if applicable):
   ```bash
   # For Node.js projects, for example
   npm install
   ```

3. Run the project:
   ```bash
   # Adjust command based on your stack
   npm start
   ```
